## 객체지향 프로그래밍 N-body Simulator 구현

## 개발 환경 
#### IDE: Microsoft Visual Studio 2017 Community Edition
#### OS: Windows 7 64bit

## 파일 설명 
#### Team17_OOP_Project : Visual Studio의 Projects 폴더입니다. 안에 Source code가 들어있습니다.
#### Team17_OOP_Proejct.exe : 실행 파일입니다. 바로 실행하실 수 있습니다.
#### Team17_Report.pdf : 프로젝트 보고서입니다.

## 실행 방법 
"Team17_OOP_Project" 폴더를 Windows 환경의 프로젝트 생성 경로에 넣은 뒤, 그 안의 Team17_OOP_Project.sln 파일을 실행하여 컴파일하면 될 것입니다. 혹시 컴파일이 제대로 이루어지지 않을 경우, "Team17_OOP_Project\Team17_OOP_Project\" 경로에 있는 CPP 파일 {main.cpp, class_func.cpp, force_tae2.cpp, simul_func.cpp}과 H 파일 {class.h, common.h, simul_func.h}을 따로 꺼내어 직접 사용하실 수 있습니다.
